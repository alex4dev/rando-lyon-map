# Randonnées sportives autour de Lyon

Carte interactive de 10 randonnées sportives accessibles depuis Lyon.

## Site

URL GitHub Pages prévue :

```text
https://alex4dev.github.io/rando-lyon-map/
```

## Contenu

- Carte Leaflet interactive.
- 10 randonnées avec traces GPX.
- Sélection d'une randonnée avec recadrage automatique.
- Interface responsive avec panneau réductible.

## Publication GitHub Pages

Dans GitHub :

1. `Settings`
2. `Pages`
3. `Deploy from a branch`
4. Branch : `main`
5. Folder : `/root`

## Maintenance

Pour tester localement :

```bash
python3 -m http.server 8080
```

Puis ouvrir :

```text
http://localhost:8080
```

La carte est un outil de préparation. Toujours vérifier météo, conditions de sentier et arrêtés locaux avant de partir.
